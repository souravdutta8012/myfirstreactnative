# myfirstreactnative
My first react native

## build with expo
expo login
expo build:android

## build with eas
eas login
eas init
eas update:configure
eas build:configure
eas build -p android (start build)
eas update (to push code to expo repo. ref: https://docs.expo.dev/eas-update/how-eas-update-works/)
